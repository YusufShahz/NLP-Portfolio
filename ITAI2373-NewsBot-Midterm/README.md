# NewsBot Intelligence System
**Course:** ITAI2373 | **Mid-Term Group Project**

## Project Overview
An end-to-end NLP pipeline that automatically processes, categorizes, 
and extracts insights from 1,440 BBC news articles across 5 categories.

## Key Results
- **Best Classification Accuracy:** 96.53% (Naive Bayes & Linear SVM)
- **Articles Analyzed:** 1,440
- **Categories:** Business, Tech, Politics, Sport, Entertainment
- **Sentiment:** 55.2% Positive, 26.4% Neutral, 18.4% Negative
- **Entities Found:** 2,301 people, 1,646 locations, 1,322 organizations
